# 學習地圖修正與驗收

## 修正範圍
- 文字：校對標題、週次說明與術語（AI、SQL、API 等）
- 圖示：週次對應的圖像是否匹配主題（準備/AI基礎/資料庫+前端+API/上線）
- 連接：階段間的箭頭與分段清楚，不跨越或重疊元素
- 版面：置中、留白、對齊一致；邊角與陰影延續現有風格

## 顯示與尺寸
- 頁面顯示：`width: 100%` + `max-width: 600px`，手機 `<=480px` 上限 `520px`，414 寬度 `480px`
- 預留尺寸：`contain-intrinsic-size: 600px 340px`，避免載入時版面抖動

## 檔案標準
- 格式：維持原始 PNG
- 色彩：RGB
- 解析度：建議保留原檔 300dpi 版，網頁端以 CSS 控制大小

## 差異與調整說明
- 先前因缺少 WebP 檔造成 `<picture>` 選擇失敗 → 已改回單一 PNG，避免載入錯誤
- 限制容器寬度避免超展：`#route .container { width: min(960px, 92vw) }`

## 驗收清單
- [ ] 桌面端不超過 600px，置中顯示
- [ ] 手機端寬度自適應；在 375/414/480 測試正常
- [ ] 無重疊、無錯位；連線與箭頭清晰
