第一部分

製作AI實用小工具

製作三個超好用的小工具，先練習對AI的操作感覺，針對程式碼與系統有進一步的認識。

第二部分

個人部落格

資料庫、前後端到權限一次打包，做出真正能用的內容平台；再把AI接上，摘要、SEO 關鍵字都幫你生好。

課程｜第一部分（2 小時）：

從頭學 AI（使用 Gemini CLI）

時數：2 小時

你將學會

  • 安裝並使用 Google Gemini CLI
  • 進階提示詞工程技巧
  • 如何在程式中呼叫 AI API
  • 實作一個 AI 小工具

必要工具

  • Python 3.10+
  • Git 2.40+
  • Node.js 18+
  • ClaudeCode（VS Code 擴充）
  • Gemini CLI / SDK

課程重點

(1)安裝設定、第一個 AI 指令、Gemini CLI 入門、API Key
(2)進階提示詞技巧、提示詞模板
(3)API 整合、使用Gemini、處理回應、錯誤處理
(4)實作專案（三選一）
  ├─ 文章摘要器：長文變 3 句話摘要
  ├─ 程式碼註解生成器：自動加上註解
  └─ 語言翻譯工具：中英翻譯＋文化說明

課程｜第二部分（4 小時）：

部落格網站（程式編輯器、會員系統、使用 SQLite 資料庫），結合前後端

時數：4 小時

你將學會

  • 完整的部落格管理系統
  • 程式編輯器串接
  • 使用者註冊、登入、權限管理

完成後你可以

  • 擁有自己的部落格平台
  • 理解內容管理系統（CMS）運作
  • 學會使用者認證與權限管理

適合對象

  • 想架設個人部落格的創作者
  • 想學習會員系統的開發者
  • 對內容管理有興趣的學員

課程重點


(1) 簡易資料庫設計   
├─ 用戶名單（管理帳號與密碼）   
├─ 文章資料庫（存標題、內文、分類）   
└─ 標籤與讀者留言區

(2) 前端網站畫面呈現   
├─ 文章首頁：清楚列出文章、好搜尋   
├─ 閱讀頁面：排版漂亮、好閱讀   
└─ 寫作工具：內容編輯器：即時預覽、草稿自動儲存

(3) 後台功能運作   
├─ 會員註冊與登入（確保身分安全）   
├─ 文章管理功能（發布、修改、刪除）   
└─ 密碼安全保護機制

(4) AI 智慧輔助應用   
├─ 讓AI自動幫寫文章摘要   
├─ 讓AI優化SEO關鍵字   
└─ 統計文章瀏覽次數統計、熱門排行

先不要因為程式碼害怕，你可以一步一步跟著我們的老師一起完成這項作品，AI簡單學這是我們一直在努力做的，就是把複雜的事情簡單化，所以試著給自己一個機會踏入你不熟悉的領域，並在多年後成為這個領域的專家，我們不必一開始就很厲害，其實只要開始，就很厲害了。

品牌故事｜簡單學AI在台灣

我們是簡單學AI團隊，我們的宗旨是幫助在這個AI的世界下，讓不太理解的人可以輕鬆與科技接軌。

團隊的每一位老師都盡心盡力的想去栽培每一位來上課的學員，在以前學程式是很辛苦的，現在有了AI的加持，對於一些刁鑽的問題，都可以很輕鬆的解決。我想要用簡單的話語來講解困難的技術，先試著喜歡，不強求要做出什麼厲害的作品，只要一步一步來，我認為現在這個時代，一個人加上AI就可以做到很多的事情，也期待有機會與你分享。希望未來可以在課堂上遇見你。
