第 1 節（30 分鐘）Gemini CLI 入門
  └─ 申請 API Key、安裝設定、第一個 AI 指令

第 2 節（30 分鐘）進階提示詞技巧
  └─ Few-shot Learning、思維鏈、提示詞模板

第 3 節（40 分鐘）AI API 整合
  └─ 用程式呼叫 Gemini、處理回應、錯誤處理

  └─ 語言翻譯工具：中英翻譯＋文化說明

課程重點

第一部分（45 分鐘）資料庫設計
  ├─ 使用者資料表（帳號、密碼加密）
  ├─ 文章資料表（標題、內容、分類、狀態）
  └─ 標籤與留言資料表

第二部分（1.5 小時）前端介面開發
  ├─ 文章列表頁：展示所有文章、分類篩選、搜尋
  ├─ 文章詳情頁：內容渲染、程式碼高亮、目錄

第三部分（1.5 小時）後端開發
  ├─ 使用者註冊與登入（JWT Token）
  ├─ 文章管理 API（新增、修改、刪除、查詢）
  ├─ 權限驗證（只有作者能修改自己的文章）
  └─ 密碼加密與安全處理

第四部分（30 分鐘）AI 輔助功能
  ├─ 用 Gemini 自動生成文章摘要
  ├─ 自動產生 SEO 關鍵字
  └─ 文章瀏覽次數統計、熱門排行
