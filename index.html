第一部分

AI 實用工具

我們會先製作三個超好用的小工具，先練習一下對 AI 的操作感覺，針對程式碼與系統有進一步的認識。

第二部分

個人部落格系統

資料庫、前後端到權限一次打包，做出真正能用的內容平台；再把 AI 接上，摘要、SEO 關鍵字都幫你生好。

課程內容｜第一部分（2 小時）：AI入門

從頭學 AI（使用 Gemini CLI），適合台灣AI初學者

時數：2 小時

你將學會

  • 安裝並使用 Google Gemini CLI
  • 進階提示詞工程技巧
  • 如何在程式中呼叫 AI API
  • 實作一個 AI 驅動的小工具

適合對象

  • 想深入了解 AI 原理的人
  • 想學會用 AI API 的開發者
  • 對 AI 應用開發有興趣的學員

必要工具

  • Python 3.10+
  • Git 2.40+
  • Node.js 18+（可選）
  • ClaudeCode（VS Code 擴充）
  • Gemini CLI / SDK

課程重點

第 1 節（30 分鐘）Gemini CLI 入門
  └─ 申請 API Key、安裝設定、第一個 AI 指令

第 2 節（30 分鐘）進階提示詞技巧
  └─ Few-shot Learning、思維鏈、提示詞模板

第 3 節（40 分鐘）AI API 整合
  └─ 用程式呼叫 Gemini、處理回應、錯誤處理

第 4 節（20 分鐘）實作專案（三選一）
  ├─ 文章摘要器：長文變 3 句話摘要
  ├─ 程式碼註解生成器：自動加上註解
  └─ 語言翻譯工具：中英翻譯＋文化說明

課程內容｜第二部分（4 小時）：台灣AI應用

部落格網站（內容編輯器、會員系統、使用 SQLite 資料庫），結合台灣AI教學案例

時數：4 小時

你將學會

  • 完整的部落格管理系統
  • 內容編輯器整合
  • 使用者註冊、登入、權限管理
  • 用 AI 自動生成文章摘要和 SEO 關鍵字

完成後你可以

  • 擁有自己的部落格平台
  • 理解內容管理系統（CMS）運作
  • 學會使用者認證與權限管理

適合對象

  • 想架設個人部落格的創作者
  • 想學習會員系統的開發者
  • 對內容管理有興趣的學員

課程重點

第一部分（45 分鐘）資料庫設計
  ├─ 使用者資料表（帳號、密碼加密）
  ├─ 文章資料表（標題、內容、分類、狀態）
  └─ 標籤與留言資料表

第二部分（1.5 小時）前端介面開發
  ├─ 文章列表頁：展示所有文章、分類篩選、搜尋
  ├─ 文章詳情頁：內容渲染、程式碼高亮、目錄
  └─ 內容編輯器：即時預覽、草稿自動儲存

第三部分（1.5 小時）後端開發
  ├─ 使用者註冊與登入（JWT Token）
  ├─ 文章管理 API（新增、修改、刪除、查詢）
  ├─ 權限驗證（只有作者能修改自己的文章）
  └─ 密碼加密與安全處理

第四部分（30 分鐘）AI 輔助功能
  ├─ 用 Gemini 自動生成文章摘要
  ├─ 自動產生 SEO 關鍵字
  └─ 文章瀏覽次數統計、熱門排行
